Acerca de Nosotros

Sistema disenado pensando en los estudiantes universitarios que desean tener un lugar en la nube especificamente para almacenar tareas y proyectos universitarios. El sistema se llevara a cabo utilizando un servidor de base de datos utilizando Oracle XE 11g, un servidor web utilizando XAMPP que a su vez funcionara como almacenamiento de los archivos de los usuarios.
